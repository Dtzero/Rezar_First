js笔记

1、javascript简介

javascript是负责页面动态部分的，

  • 用户与页面之间的动态交互
  • 服务器与页面的动态交互
  • 能够更改页面中得结构样式

javascript通过script标签链接到html中，有以下三种方式

  • 内嵌链接
  • 行间链接
  • 外联链接

script标签在html中的位置

  • head结束前
  • body结束前

2、变量

在javascript中，我们通过var声明变量，例如 var 名字

注意：变量的命名规则为，变量名只能由数字，字母，下划线，美元符组成，首字母只能为字母，下划线，美元符。数字不能作为变量名的起始字符，以便于javascript能区分标识符和数字；

3、事件

事件是可以被javascript监测到的行为，通俗来说就是各种交互时的行为，每个交互的一瞬间都可以产生一个事件，一般我们将事件和函数配合使用，当发生事件时执行函数；

4、函数

函数是由事件所引发或者当它被调用时能够执行的代码块，javascript中声明函数语法如下

function 函数名（参数）{
    函数体或者叫代码块
}

我们还可以利用函数表达式来创建一个函数

var 名字 = function 函数名（参数）{
    函数体或者叫代码块
}

若不加函数名则创建了一个匿名函数，匿名函数只能利用函数表达式来创建，如果利用声明函数来创建一个匿名函数会报错；

函数调用：目前所知函数调用方法为直接调用，即直接书写函数名；

5、属性调用

我们可以通过.和[]来进行属性的操作，例如bg.style.width='200px';

6、元素属性

元素中的某些属性是可以进行读写的，目前已知一下几种

  • id名
  • class名
  • value属性
  • innerHTML结构
  • src属性
  • href属性

7、字符串拼接

我们可以通过+号来对两个字符串进行拼接,用于字符串则表示拼接前后两个字符串，例如

"miaov" +"ketang" = "miaovketang"
"ketang"+"miaov"  = "ketangmiaov"
字符的顺序参照加号的位置
